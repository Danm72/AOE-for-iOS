Welcome to Age of Empires for iOS.

To install this game on a device you require an Apple Developer license or a jailbroken phone.  

$ git clone : https://github.com/Danm72/AOE-for-iOS.git

This clone can be preformed in Xcode, and the project will be setup by the IDE.    The project can be deployed to a choice of device or simulator 

WHY!?

This project was created as a learning exercise, and to encourage others with the same area of interest to think big.
